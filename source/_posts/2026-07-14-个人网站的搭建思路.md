## 为什么想搭个人网站

我一直想要一个属于自己的空间，可以整理和沉淀：

- **想法**：日常思考、读完书/看完电影后的感想
- **学习**：正在学的东西，尤其是资产评估师备考过程中的笔记
- **消费**：中到大宗的消费记录，并且可以看到"这个东西买了 X 天，平均每天花了多少钱"

相比于发在社交媒体上，个人网站不会沉底，不需要担心平台的算法和审查，所有数据都在自己手里。而且，Frutiger Aero 风格真的很漂亮。

## 技术方案

我对网站搭建一窍不通，所以把这个任务交给了 Claude。最终选择了 **Hexo** 这个静态博客框架，搭配一个 Frutiger Aero 风格的主题。

Hexo 是一个 Node.js 工具，用 Markdown 写文章，一条命令编译成纯 HTML 文件，免费部署到 GitHub Pages。整个流程非常轻量：
1. 在本地用 Markdown 写文章
2. 运行 `hexo server` 预览
3. `git push` 推送到 GitHub
4. GitHub Actions 自动部署，30 秒后网站更新

域名先用免费的 `github.io`，等网站内容丰富之后，再考虑买一个 `.com` 或 `.top` 域名绑上去。
